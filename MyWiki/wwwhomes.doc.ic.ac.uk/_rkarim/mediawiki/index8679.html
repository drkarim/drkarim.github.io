Locked out

From Rashed Karim Wiki

Jump to: navigation, search

Just recently I was locked out of my windows XP machine. This happened as my computer used to part of a network, and now it's just a standalone PC. I removed it from the domain and made it part of my own home WORKGROUP. Most standalone PCs are part of a default workgroup. Changing workgroup triggers the system to ask for a restart, which I did and I was locked out.

I didn't have the windows CD and thought that was the end of it. Here is what I did using a nice piece of software I found on the web:

  • Downloaded this file publicly available here
  • Burn the ISO on a CD, using your favourite CD burning utility. Make sure you select Burn image. Note: this is not like burning music on a CD, or copying other data to CD.
  • Boot from the CD
  • The program boots the system in linux and loads the NTFS partition (if you have many partitions);.
  • At this point, you pretty much follow instructions on the screen. However, sometimes the NTFS may load only as Read-only in which case, all the changes you make (such as password resets) will not work. Restart windows in Safe mode and make sure the PC is shut down properly to disable the read-only lock.

Note: Only use this if you are locked out of your 'own' PC
