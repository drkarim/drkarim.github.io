VTK points display on paraview

From Rashed Karim Wiki

Jump to: navigation, search

Issue

A list of points with x y z doesn't display on Paraview unless you have them in a correct format in the vtk file.

Keywords

vtkPoints paraview vertices vtk

Solution

Fix the format and add the vertices list which is mostly omitted unknowingly. For example if a points list has 45 points, the line defining the start of vertex list should be: 

VERTICES 45 90

where 90 is 45 x 2. In the VTK file format definitions found here this should field should indicate the size.

Here is a VTK points list that works and displays correctly in paraview:
