Processing pipeline

File location

  • Dropbox folder -> tob_collab_kit
  • Run VisualizeCaseResults on matlab (Open VisualizeCaseResults.m)
  • Press the play button to run it. This will clear memory, etc.
  • Manually execute all the different sections in code:
    • Code divided into cells, separated by horizontal lines
    • Select -> Rightclick on each section (cell) -> Evaluate current cell
    • Run first two sections - 1) Load data (loads the electrical data into matlab var), 2) save UnfoldInput.vtk (mesh in vtk with vertex point id as scalar)
    • Offline unfold
      • Remember when unfolding, no new vertices are added. This is important for the remaining of the steps to work
      • This is achievable in new version of paraview. Use plane clipping (filter) and crinkle option.
    • Store unfold result as UnfoldOutput.vtk
      • For each vertex in unfold, store the 3D mesh vertex ID it corresponds to
    • Now run the other sections in the code, which will load the UnfoldOutput.vtk and generate the result vtk files that are ready to be loaded into paraview for animation


16 April 2014

  • Novelty in their technique:
    • Standard approach, using a catheter with circular electrodes, record electrogram in each catheter, see signal waveform, choose a time window and represent the propagation on 3D. This can be done using velocity system.
    • the novelty: No need to choose time window, represent energy of EGM (Weber et al. TMBE 2010).
  • In video of wave propagation (shown on unfold earlier), the flow are signals acquired using the circular catheter at different locations (covering the LA). Reconstructing the flow of local activation pattern (represented as its energy). So red regions in that video show high energy.
  • The advantage of using energy as opposed to say amplitude of signal is that there are very small differences in amplitude, fractiation, etc. So in a video of amplitude, there is not much amplitude to visualize
  • Next steps:
    • Look at two Atrial flutter cases, extracted from velocity, represent on circular unfold
    • Clinicians expressed concern about distance between MV and APP as this was too large compared to the roof. Perhaps MV cut was not precise. Provide 3D shells with cut so T can check MV cut.
