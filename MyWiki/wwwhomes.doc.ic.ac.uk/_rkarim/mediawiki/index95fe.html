Notes from these lectures

An example of a non-uniform B-Spline curve with n=6 and k=3. Non-uniform since each control point does not influence exactly k=3 curve segments. For example the first control point only influences the first curve segment.
  • Disadvantage of Bezier curves - With increasing control points, the order of the curves also increasing. So if we chose 10 control points, we would have terms such as (1-u^9). For example, 3 control point give a quadratic Bezier curve. This is not the case with B-Spline curves, where the number of control points is independent from the order of the curve. The order of the curve can be changed by specifying the value of k.
  • Advantage of B-spline curve - it has local propagation, in Bezier curves changing any control point will change the entire curve. This is global propogation. Whereas B-Spline curves are defined in terms of many curve segments (all meeting continuity requirements). So if say n=10 (control points) and k=4, and you change one of the control points, the entire curve doesn't change, it is only a local portion of the curve (close to the control point you have changed) which changes.
  • B-Spline curves are defined in terms of n and k (control points and order). The number of curve segments are n+k-2. So if n=6 and k = 3, curve segments = 5.
  • If n=6, k=3 and there are five segments - every segment is influenced by k=3 points (around that segment).
  • In the same way as above, the control points also influence the individual curve segments. In the figure, the influence of each control point over curve segments is not uniform: For example, the first control point only influences first segment. Third control point influences the first, second and third segments. This non-uniformity of influence over curve segments makes the B-spline non-uniform. For uniform B-Spline curves, the curve does not pass through the first and last control points. However, as it is commonly a requirement for B-Spline curves to pass through the first and last control points, non-uniform B-Splines are more popular.
  • k=4 is a popular choice for B-Spline curves - gives slop + curvature continuity (C^3 continuity).
