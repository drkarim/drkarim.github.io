Local regions

  • To obtain local regions (or bounding boxes) from STAPLE segmentations, staple was thresholded at 70 converted to gipl and then using this bounding box program for connected components in the image, the coordinates of the bounding box of each segented region was found. The magic2 code was run to generate a connected component image.
  • Script to do this:

(Found in C:\PhD\experiments\kcl_data\isbi_seg_challenge_upload\dice\local_dice_pre\Detecting_local_regions as GenerateAllLocalRegions.bat).

Note magic2 worked in 20 out of the 30 datasets. 

for /l %%n in (1,1,30) do ( 
	IF EXIST  C:\PhD\experiments\kcl_data\isbi_seg_challenge_upload\staple_manual_segs\staple_gipl_at_threshold_at_70\staple_a_%%n.gipl (
		
		magic2 -i1 staple_a_1.gipl -o concomp.gipl --concomp x 
		concompbox -i1 concomp.gipl -o AllLocalRegions_p_%%n.txt 
		
	) 
)
  • All regions in AllLocalRegions_p_XX.txt copied into Excel file AllRegionsSorted.xls
  • The regions were sorted (first by case number), and then by size using excel custom sort button.
  • Regions with a significant size were chosen from each case
  • These regions individually copied over to C:\PhD\experiments\kcl_data\isbi_seg_challenge_upload\dice\local_dice_pre\LocalRegions
