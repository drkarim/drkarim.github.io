Generate Shells

  • Two shells are necessary 1) Shell with ground truth, 2) Shell with challenger's segmentations
  • To generate shell with ground truth use the #Ground truth shell generation script
  • The shell with challenger's segmentation are generated on the fly in the #Challenger RMSE generation script (See scar3d_v6 command)

Generate RMSE

  • RMSE's were generated for each segmentation using #Challenger RMSE generation script for each challenger. These are found in C:\PhD\experiments\kcl_data\isbi_seg_challenge_upload\rmse. The RMSE C++ program can be found here.

Analyse RMSE

  • Put all the RMSE in one text file all_rmse.txt and this can be found in C:\PhD\experiments\kcl_data\isbi_seg_challenge_upload\matlab\post_analysis
  • The mean and Stdev were computed using the #RMSE Mean script below.

RMSE Mean script

  • MATLAB script to compute Mean and Stdev by ignoring NaN where datafile is all rmse in one file. This file can be found in C:\PhD\experiments\kcl_data\isbi_seg_challenge_upload\matlab\post_analysis 
function [means, stdev,R] =ProcessRMSE(datafile)
    R = load(datafile); 
    I = R <= 0;
    R(I) = NaN;
    
    for i=1:size(R,2)
        T = isnan(R(:,i)); 
        means(i) = mean(R(~T,i));
        stdev(i) = std(R(~T,i));
    end
end
