INSTALLATION

  • Copy folder from USB
  • Add to system path

SEGMENTATION

  • Dcm2nii converter can be downloaded from:

http://www.mccauslandcenter.sc.edu/mricro/mricron/dcm2nii.html

  • Segment 3d-wh using ITK-snap
  • Re-name the 3d-wh segmentation and le as: 3dwh.nii and le.nii
  • Make sure you put these in a folder on windows. Let’s call it images_temp
  • Measure healthy (BP) and scar tissue means and SD and compute the ratio.
  • Open command-line prompt, change directory to the image folder: cd images_temp
  • Copy preprocess.bat (included in the software) to the images_temp folder
  • Type: preprocess <1+mean ratio> <SD ratio> in the command prompt
  • Type: segment <1+mean ratio> <SD ratio> in the command prompt
  • Select _gc_out_seg_3 or 4 (check on rview loading le as target and _gc_out as source) Use Iso-lines (of source) to bring up the contour. Move threshold slider to reveal contour around scar.
  • Type postprocess <selection of gc_out whether 3 or 4> <10> - just check if number of connected components is greater than 0 (means its ok)
  • Need to convert to GIPL: 
nii2gipl
  • To visualize on the shell, in command-line type: scar3d –w .
  • In command-prompt asks for three questions: Put 1 <enter> 0 <enter> 1 <enter>
