Interpreting surface ECG

Quick introduction to EKG, i.e. electrogram interpretation

  • Depolarization is an electro-chemical process within the cardiac myocytes. Depolarization causes contraction of ventricle and atria. Depolarization starts at SA node.
  • The EKG nomenclature is important. In a single beat, in the EKG, there are P, QRS and T-waves:
    • P-waves correspond to atrial contraction (or depolorization)
    • QRS is ventricular contraction (depolarization)
    • T-wave represents ventricular repolarization
  • PR interval
    • Distance between the P wave and start of QRS complex:
    • indicates AV conduction time, varies between 0.1 and 0.2 secs. Longer intervals are found with increased heart rate.

MRI images dicom matrix

  • Dicom patient position information
    • In MRI images the corner or origin of the image stack is stored as xyz in the header
    • This is in the coordinate system of the scanner
    • This coordinate system is centered at the centre of the magnet, and all points and positions are relative to that
    • There is also the xy direction stored in the dicom header as image orientation
    • When dicom images are converted to niftii or gipl, this information is converted into a matrix form and stored in the image header of the 3d image
    • however, using kcl image conversion legacy tools, this matrix is stored in a separate file
    • This I have been referring to scanner matrix or by other names, is used to register or orient one image to the other from the same scanner.
