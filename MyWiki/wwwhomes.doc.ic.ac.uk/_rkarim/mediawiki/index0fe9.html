Issue

Add a claim to an invoice after generation

Creating or modifying entries

Claims

  • You need to add the new claim to the claims table. For this, first extract this information:
    • Get student ID, lets call this S
    • Get tutor ID, lets call this T
    • Get student invoice ID, lets call this SI (This is the invoice you are adding to)
    • Get the tutor pay rate for this claim, call this TP. This you can find TP from student_tutor table in the hourly_Rate_to_tutor column (get latest rate, not old expired rates)
  • Now you are ready to add the claim, Go to the claims table and create this entry:
    • Claim_id = (automatic assignment), tutor_id = T, student_id = S, session_date = (you know), hours = (you know), session_Type_id = 0, awaiting_approval = 0, approved_on = (before or on invoice generation date), student_invoice_id = SI, claim_amount = TP x hours taught
  • Once you have inserted this claim, determine the automated assigned claim id, lets call it C

Invoice_to_claim

This is the tutor invoices. Note the invoice id numbers here DO NOT correspond to student invoice numbers.

invoice_to_claim and invoice_to_month are two tables storing the data. To determine which invoices belong to which tutors, the claim_id in invoice_to_claim and claims table is matched (join query)

  • If this tutor has put a claim this month, determine which are the other claims this month and create new entry:
    • find their claim no., lets call this Cx
    • In invoice_to_claim, find invoice number corresponding to Cx. Lets call this TI. Double-check to see whether TI is pointing to other claims in the same invoice, as it should do.
    • Add the entry invoice_number = TI and claim id = C (Not Cx).
  • If this tutor hasn't put a claim this month, create a new entry in invoice_to_claim:
    • In invoice_to_claim, go to the end of table, and find the last sequentially assigned invoice_id, lets call it TI.
    • Add the entry: invoice_number = TI+1' and claim id = C
    • Important: Add an entry to invoice_to_month, with TI+1 and date as the invoice date

Student invoice

  • GO to the invoice entry with SI and modify the invoice total by adding the amount-client-pays rate. You can get this rate from student-tutor table.

Final checks

To ensure everything is consistent:

  • Check tutor invoices to see if the tutor who put the new claim has the correct updated invoice.
  • In the student's invoice, check new entry and invoice total
