Number formats for setting the color legend

Paraview's colorbar legend range and number formatting follows the same rules as C uses to format to output characters and numbers using stdout

Here is a setting I used which enabled me to label data between [0.58,4.3] to an integer colour scale. The settings I used were:

Settings in paraview needed for an integer labelled colour map.
  • %-3.01g for label format
  • %2.0g for range label format

It follows the C++ convention which is as follows: [flags][width][.precision][length]specifier
