Baatch file for generating t2 shells

From Rashed Karim Wiki

Jump to: navigation, search

Code

  • This is code I wrote for Y for generating a t2 shell out of hand segmentations of T2. It assumes that certain files are there in the folder on which it executes. For example: 3dheart, 3dheart_seg_LA, t2.
  • It takes one parameter as input, which is the pixel value in the mask (hand segmentation) that indicates T2 signal corresponding to Oedema. For example, 1 for Oedema and 2 for normal atrial wall. 
transformation t2_handseg_YA.gipl t2_handseg_YA_in_3dheart.gipl -dofin t2_2_3dheart.dof -target 3dheart.gipl 
transformation t2.gipl t2_in_3dheart.gipl -dofin t2_2_3dheart.dof -target 3dheart.gipl 
magic -i2 t2_in_3dheart.gipl -i t2_handseg_YA_in_3dheart.gipl -o temp.gipl --ms x -t 1 
threshold temp.gipl temp2.gipl %1
scar3d -i2 temp2.gipl -i1 3dheart_seg_LA.gipl -m 0 -s 1 --ben x -o t2_handseg_shell.vtk
