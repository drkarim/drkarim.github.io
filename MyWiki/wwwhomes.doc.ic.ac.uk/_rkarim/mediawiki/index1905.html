Plotting histogram of a GIPL image

  • Load GIPL image into memory and use the hist command 
I = giplread('<path to file>'); 
I = I(:);   % 3D vector becomes 1D vector 
hist(I, 255); % 255 is number of bins
  • If the background is yielding a high frequency, discard background: 
nz = I>0; 
I = I(nz); 
hist(I, 255);

Plotting histograms of an image

Note we re-scale the histogram frequency (see below n_norm and n_2). n_norm normalizes the histogram between 0 and 1. n_2 re-scales the histogram to that of the Gaussian mixture curve. So the 0.02 factor is the maximum of the Gaussian mixture curve. Normally, this value can be determined by looking at the plot generated from the EM command: EM_GM(A,4,20,[],1,[]), which draws the individual and combined gaussians following the EM computation. 

A=load('_dilated_seg_hist.txt'); 
n=0:400; n=hist(A,x); n_norm=n/max(n);
[W,M,V,L] = EM_GM(A,4,20,[],1,[]);
n_2 = n_norm*0.02;
hold on
stairs(x,n_2);

Plotting a Histogram from a file containing the values

Assuming that the values to be represented into the histogram are found in a text file. x denotes the x-axis or the bin intervals. hist bins the elements in A according to x.

Commands 

A=load('intensity.txt');
>> x=0:1000; 
>> n=hist(A,x);
>> bar(x,n);

If you wanted a line-graph of the histogram, if D is where the data is 

[y,x] = hist(D, 50)
plot(x,y)

Custom-script for Histogram 

%   [C]=AutoHist(file1, uLimit, bins, fs)
%   file1 - filename of the intensity values to be plotted into histogram 
%   uLimit - uLimit for normal data, anything above is outlier and thus
%   discarded
%   fs - optional parameter for font size
%   Rashed Karim , Imaging Sciences, KCL, London rashed.karim@kcl.ac.uk
function [C]=AutoHist(file1, lLimit, uLimit, bins, fs, filename)
    C=[0];
    if nargin < 5
        fs = 16
    end
    figure1 = figure('Color',[1 1 1]);
    A = load(file1); 
    for i=1:size(A,1)
        if A(i,1) < uLimit & A(i,1) > lLimit
            C(1+size(C,1),1) = A(i,1);
        end
    end
    hist(C,bins)
    set(gca, 'FontSize', fs);
    set(gca,'XLim',[lLimit uLimit]);
    ylabel('Frequency', 'FontSize', fs);
    xlabel('Pixel intensity', 'FontSize', fs);
    
    if filename ~= -1
        print(figure1,'-djpeg', filename)
    end

end
