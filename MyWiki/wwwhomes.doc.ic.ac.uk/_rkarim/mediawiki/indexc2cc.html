Experiment

  • Place sensor at three different locations on the catheter - base, middle and tip.
  • Move along X+ one step at a time, record XYZ in file
  • Note start is fixed.

Experiment details

  • Setup server
  • Run ven2
  • Load model and registration matrix
  • Connect to server
  • Press X+ one step at a time. Positions are being recorded in received_xyzs vector
  • Press Reg Pos button, asks for filename, specify file, positions are written from received_xyzs to specified file (see WritePositionsToFile_cb function)
  • Press X reset button
  • Go to file and leave some new lines to distinguish next incoming positions
  • Change sensor position to middle or tip
  • Go back to X+ move step
