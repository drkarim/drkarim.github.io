Landmark registration

To register the co-ordinate systems of the Aurora magnetic coil tracker and the 3D glass heart, physical lead ball landmarks are placed in the glass-heart and CT scanned.

  • Landmarks appear as bright pixels in CT scan.
  • Can be visualised clearly on CT MIP - use scar3d to visualise shell with MIP (landmarks are red dots). Save shell as VTK
  • Modify code in rkapp1 -w . to output points selected. Remove comment from: 
void keyPressedFunction_window2(vtkObject* obj, unsigned long,void *sr, void *): 
points_out << vert_xyz[0] << "\t" << vert_xyz[1] << "\t" << vert_xyz[2] << endl;

In main function: 
points_out.open("points_selected.txt", ios::app);

In header: 
ofstream points_out;
  • Open rkapp1 -w . (landmark + ICP) registration software and open above VTK shell
  • Select points using '2' and this writes to a text file named points_selected.txt.
  • Now in the Robotic controller software, open this landmark file to display the landmarks.
