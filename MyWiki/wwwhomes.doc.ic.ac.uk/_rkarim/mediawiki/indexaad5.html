Introduction

This page contains instructions for setting up SSL on Apache2. Here are some key points:

  • There is no setup necessary at least in the php code unless all your files and JavaScript and CSS have relative links, I.e. no hard coded http.
  • the setup is performed on the application layer. Installation of certificates, etc.
  • Certificates to be purchased from the cruciate authority that is well known such as GeoTrust or those verified by Mozilla, Microsoft or Google.
  • If it is renowned CA then the browser wing alert user that on the browser
