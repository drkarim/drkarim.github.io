Code 

echo Let me run rigid registration 
rreg %1 %2 -dofout rreg.dof 
echo ****************************
echo *	 
echo * OK, now I am running 
echo *
echo *
echo ****************************
areg %1 %2 -dofin rreg.dof -dofout areg.dof  
transformation %2 %4 -dofin areg.dof -target %3
threshold %3 %5 %6

Example usage 

a_2_b MRA_post.nii LE_post.nii CJ_seg_post.nii post_le_in_post_gad.nii post_gad_seg.nii <threshold for post CJ segmentation>

And for example in pre data, this is what we would use: 

 
a_2_b MRA_pre.nii LE_pre.nii CJ_seg_pre.nii pre_le_in_pre_gad.nii pre_gad_seg.nii <threshold for pre CJ segmentation>

Where to find

This file is stored somewhere where it is included in PATH variable on the Mac windows bootcamp.

Second batch program

There is yet another batch program. This I have called as Louisa. Here is the batch code: 

echo **************************************
echo
echo Registering the pre and post gad scans
echo 
echo **************************************
echo
rreg MRA_post.nii MRA_pre.nii -dofout pre_mra_2_post_mra_rreg.dof 
areg MRA_post.nii MRA_pre.nii -dofin pre_mra_2_post_mra_rreg.dof -dofout pre_mra_2_post_mra_areg.dof 
echo
echo
echo **************************************
echo
echo Registering the pre LE scan to pre gad scan
echo 
echo **************************************
echo
rreg MRA_pre.nii LE_pre.nii -dofout pre_le_2_pre_mra_rreg.dof 
areg MRA_pre.nii LE_pre.nii -dofin pre_le_2_pre_mra_rreg.dof  -dofout pre_le_2_pre_mra_areg.dof   
echo
echo
echo **************************************
echo
echo Generating pre_le_in_post_gad scans
echo 
echo **************************************
echo
transformation LE_pre.nii pre_le_in_pre_gad.nii -dofin pre_le_2_pre_mra_areg.dof 
transformation pre_le_in_pre_gad.nii pre_le_in_post_gad.nii -dofin pre_mra_2_post_mra_areg.dof -target MRA_post.nii 
echo
echo
echo **************************************
echo
echo Registering the post LE scan to post gad scan
echo 
echo **************************************
echo
echo
rreg MRA_post.nii LE_post.nii -dofout post_le_2_post_mra_rreg.dof 
areg MRA_post.nii LE_post.nii -dofin post_le_2_post_mra_rreg.dof  -dofout post_le_2_post_mra_areg.dof 
echo
echo
echo **************************************
echo
echo Generating post_le_in_post_gad scan
echo 
echo **************************************
echo
echo
transformation LE_post.nii post_le_in_post_gad.nii -dofin post_le_2_post_mra_areg.dof -target MRA_post.nii
