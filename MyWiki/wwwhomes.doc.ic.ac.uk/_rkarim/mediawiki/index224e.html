Please note that this code does not compute the Bland-Altman numbers, it only plots the off-line generated bland-altman numbers to a scatter plot.

Steps

  • Save the following code as a .m file.
  • Point MATLAB workspace to folder containing this file
  • Open the file from matlab (file -> Open)
  • Run the file (run or play button in code editor)
  • Empty plot window will appear
  • Now, press drop-down beside run/play button (It is called the 'Edit run configruations' option)
  • Now enter the input arguments to create figure
  • Here are the input arguments 
createBlandAltmanFigure(avg, diff, [5,40], [-1.5,-1.5], [11,11], [-14,-14])
  • Here avg - the column vector containing the bland-altman average( i.e. (A+B)/2 )
  • diff - the column vector containing the bland-altman difference (A-B)
  • The remaining arguments are for the drawing horizontal dashed lines denoting the mean and 95% limits of agreement [5,40]. See here and the wiki article on these plots.
  • The first argument is XData1, this is just the x range for the lines. So put (5,40) to start with and you can edit this later in property editor.
  • The second argument is YDATA1, this is the line denoting the mean (i.e. mean difference). Compute these values off-line and if this value is m, put as [m,m].
  • The third and fourth arguments YData2 and YData3 are the +2SD and -2SD lines respectively. Again compute these off-line and put as [x,x] and [y,y] if x and y are the computed values.

Note that this .m file was generated after I set the properties for the bland-altman plot, and generated matlab code for these settings, File -> Generate code (In figure). See the matlab guide for reproducing figures here.
