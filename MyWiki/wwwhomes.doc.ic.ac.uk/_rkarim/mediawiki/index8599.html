Instructions

  • Open scar3d software, this is an old version of scar3d found in
  • Ctril+1 - load 3dheart segmentation
  • Ctrl+2 - load 3dheart segmentation (again)
  • At the prompt, enter 1 and then 0 (mean) and then again 1 (stdev)
  • You will see the LA shell displayed at this point
  • Select the LA using a box. press 'x' to open the bounding box
  • Tools -> Compute LA centroid (this computes the centroid of the LA)
  • Now select two points using 'h' on the shell - the first is usually the septum and the other one is the PV
  • If you want to stick the same septum location, select a different PV using 'h'
  • If you want to change septum location, hit the 'reset PV angle' button and press 'h' to select septum and then vein.
