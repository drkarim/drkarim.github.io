Introduction

Segmentation

  • Hyon-mok was provided with scan2.gipl. This can be found in phani-pc\students\hyon-mok\ folder.
    • The segmentation performed can be found in: \hyon_mok_segmentations_of_phantom sub-folder
    • Full_Segmentation_30June14_With_borders is the original segmentation
    • Full_Segmentation_30June14_With_borders_with_blurring is the post-processed blurred segmentation
  • Since he was provided with the old scan, the new scan was registered and then transformed as phantom2_reflected_in_scan3_areg.gipl. This scan is registered to scan2.gipl and thus can be overlaid correctly on Full_Segmentation_30June14_With_borders_with_blurring or Full_Segmentation_30June14_With_borders

Surface model generation

  • New segmentation stored in phani-pc\students\hyon-mok\ folder
  • This is called XyoXXXk_segmentation_for_generating_surface. This was generated after rescaling the segmentation to [0,100] and then blurring with sigma-1 and then thresholding at 30.
  • Surface can be generated using two methods:
    • Without projecting landmarks. This uses surface.exe.
    • Project landmarks. This uses scar3drobot.exe. Note this is the step used to generate surface
  • The surface can be found in phani-pc\students\hyon-mok\ as phantom_with_borders_HMO.vtk


Calibration experiments

  • After you have calibrated, you should save the .rgx registration matrix file. Assuming we will be doing various calibrations, we repeat this step.

Once calibration is completed:

  • open software (ven2.exe), load new mesh, load registration matrix
  • select 'track target distance'.
  • position the tracker at a landmark, click 'track'
  • click 'x' to tell software where you think the point should be (green dot)
  • Repeat step 3,4 for a set of landmarks
  • rename and save 'selected_positions.txt'
