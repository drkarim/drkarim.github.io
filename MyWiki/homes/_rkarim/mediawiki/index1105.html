Initialization

When working on the robot, these are the steps required to be performed before the catheter can be controlled using model software:

  • Registering Physical and Model space: Landmark registration
  • Test: Tracking robot position
  • Catheter Calibration: Calibrating catheter

Experimentation

  • Starting and stopping the robot and client connections
  • Initialize robot and staying limits of stepper motors
  • Simple experiment to measure time to step relation in X and Y motors revealed this as T = 0.2 x n + 0.5, where n is number of steps and T secs.
  • Forward kinematics of robot
  • Work with student on evaluating accuracy of calibration

Control

  • Control robot to target location
  • Tracking aurora software, C-sharp written by Ma and example code available in dropbox Ali_robot_code folder
