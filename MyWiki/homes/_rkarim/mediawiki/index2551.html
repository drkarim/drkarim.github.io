Robot control

Software instructions

  • Start the robot software. On mac laptop path is C:\PhD\software\Venus3D\bin for software project and ven2.exe
  • File -> Load LA Model (path in mac-laptop: C:\PhD\experiments\kcl_data\robot_phantom\tests_on_new_phantom_scan\model_no_ventricle_with_landmarks )
  • Connect to Robot
  • Registration -> Load registration matrix -> matrix.rgx. If registration is already completed in this current setup. If not, then follow the registration steps.
  • [Calibrate catheter]
  • Start control (Press the control button)

Code documentation

  • Once control is started, mode 50 is activated.
  • In this mode, the 'x' button when pressed performs the following actions (see keyPressEventHandler function):
    • Converted selected Model point to physical co-ordinates, using inverse reg. matrix 
	// control mode - send physical co-ordinates to robot 
	// so first convert model to physical space using the inverse transformation
	Point3D p(pick_position);
	Point3D pT;
	Physical2Model.applyInverse(p, pT);
    • Send these co-ordinates across to robot: 
sm << "TARGET," << pT[0] << "," << pT[1] << "," << pT[2];
System::String^ sendSm = gcnew System::String(sm.str().c_str());
Communicator::SendMessage(sendSm);
