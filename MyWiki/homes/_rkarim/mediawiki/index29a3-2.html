Sirius change pay rates

From Rashed Karim Wiki

Jump to: navigation, search

Usage

  • Change pay rate for a single client for all his/her existing students.


Important considerations

  • If there are claims that should be retrospectively changed due to the updated rate, the following script does not update these

Scripts

  • Script location: orphan_scripts/UpdateTutorRateForHisAllClients
  • The following variables must be set before executing the script 
/*
	*	USER MUST ADUST THESE PARAMETERS BELOW 
	*/
	$tutor_id = 20;
	$new_hourly_rate_to_tutor = 35;
	$rate_effective_from = "2014-03-31";
